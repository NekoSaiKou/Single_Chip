## 可以演奏的小棒棒  Play music with a stick

### Platform  
*   Arduino
*   MPU6050

### Prototype
*   Buzzer and arduino tone library

### User Guide
*   C --swing left
*   D --swing top left
*   E --swing top
*   F --swing top right
*   G --swing right
*   A --swing bottom right
*   B --swing bottom
*   C --swing bottom left
